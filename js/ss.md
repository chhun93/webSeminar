ES, JS
===========
#### Script
독립된 시스템에서 작동하도록 특별히 설계된 프로그래밍 언어   
사용자 의도에 따라 동작 가능

#### ECMA
정보 통신에 대한 표준을 제정하는 비영리 표준화 기구   

#### ES (ECMA Script)
ECMA 인터내셔널에 의해 제정된 기술 규격에 의해 정의된 범용 스크립트 언어
현재 ECMA-262 표준을 사용(ES6)   

#### JS (Java Script)
ES 사양을 준수하는 범용 스크립트 언어

#### 차이
ECMAScript 문서를 읽게 된다면, 어떻게 스크립트 언어를 만들 수 있는지를 알 수 있습니다. JavaScript 문서를 읽게 된다면, 어떻게 스크립트 언어를 쓸 수 있는지를 알 수 있습니다.
